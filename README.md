```
│   └── ...

│   └── 板块问题.tex
├── 英语/              # 英语笔记

```

## 🛠️ 编译说明

本仓库使用 **XeLaTeX** 编译，支持中文。

### 基本编译命令

```bash
xelatex 文件名.tex
```

### 推荐编译流程

为了获得正确的目录和交叉引用，建议编译两次：

```bash
xelatex 文件名.tex
xelatex 文件名.tex
```

仓库中提供了 `数学笔记模板.tex` 和 `数学笔记模板使用说明.md`，可以参考模板创建新的笔记。
